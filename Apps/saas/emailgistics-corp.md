#### <a name="data-access-using-microsoft-graph"></a>Доступ к данным с помощью Microsoft Graph

>|   **Graph разрешения**  | **Тип разрешения** |          **Justification**          | **Идентификатор приложения Azure AD** |
>|:------------------------|:--------------------|:------------------------------------|:--------------------|
>| Mail.ReadWrite | Делегированные | Прочтите req'd доступа, чтобы получить сведения о сообщениях в общем почтовом ящике. Сведения о сообщении необходимы для выполнения определяемых клиентом правил (например, для правильного отслеживания бесед (для журнала бесед и автоматического назначения) и для создания отчетов (например, отчета об открытии сообщений), если сообщение содержит &#8220;счет&#8221;).
Напишите запрос на доступ для отправки сообщений из правил (например, автоответов) и для обновления тем сообщений с помощью идентификатора беседы emailgistics. | [2c9fb9b7-5112-4a91-af52-f98682bc7bf3](../azure/2c9fb9b7-5112-4a91-af52-f98682bc7bf3.md) |
>| Mail.Send | Делегированные | Позволяет emailgistics отправлять сообщения электронной почты из общего почтового ящика. Это разрешение требуется для отправки сообщений из правил (например, автоответов). | [2c9fb9b7-5112-4a91-af52-f98682bc7bf3](../azure/2c9fb9b7-5112-4a91-af52-f98682bc7bf3.md) |
>| MailboxSettings.ReadWrite | Делегированные | Позволяет emailgistics получить имя общего почтового ящика и создать правило почтового ящика для маршрутизации входящих сообщений в скрытую папку входящих сообщений для начальной обработки, выполнения правил и автоматического назначения. | [2c9fb9b7-5112-4a91-af52-f98682bc7bf3](../azure/2c9fb9b7-5112-4a91-af52-f98682bc7bf3.md) |
>| User.Read | Делегированные | Позволяет emailgistics выполнять вход в качестве общего почтового ящика и читать основные сведения о нем, такие как идентификатор субъекта-службы, адрес электронной почты и имя пользователя. | [2c9fb9b7-5112-4a91-af52-f98682bc7bf3](../azure/2c9fb9b7-5112-4a91-af52-f98682bc7bf3.md) |
>| User.ReadBasic.All | Делегированные | Позволяет emailgistics читать основные сведения о пользователях в клиенте. Требуется для обнаружения изменений в адресах электронной почты и именах участников почтового ящика и автоматического обновления этой информации, которая используется для отслеживания и создания отчетов, в emailgistics. Это единственным разрешением, область действия которого выходит за пределы общего почтового ящика. | [2c9fb9b7-5112-4a91-af52-f98682bc7bf3](../azure/2c9fb9b7-5112-4a91-af52-f98682bc7bf3.md) |
>| offline_access | Делегированные | Позволяет emailgistics работать в качестве серверного приложения, а не интерактивного браузерного приложения, требующее, чтобы пользователь вход в систему, когда он работает. Это разрешение позволяет emailgistics получить маркер обновления, который может обновляться при необходимости для постоянного доступа. | [2c9fb9b7-5112-4a91-af52-f98682bc7bf3](../azure/2c9fb9b7-5112-4a91-af52-f98682bc7bf3.md) |
